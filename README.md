# <h3>🧠 Avaliação de Saúde Mental Organizacional</h3>

Este projeto consiste em uma aplicação web voltada para avaliação de saúde mental em ambientes organizacionais, 
com foco em usabilidade, interface moderna e práticas de desenvolvimento front-end.

# <h3>🚀 Funcionalidades</h3>

<li>🔐 Sistema de login e cadastro de usuários</li>
<li>🧾 Armazenamento de credenciais no Local Storage</li>
<li>🔒 Aplicação de criptografia básica para proteção de dados</li>
<li>📊 Interface para avaliação de saúde mental organizacional</li>
<li>📂 Sidebar interativa para navegação</li>
<li>💻 Layout moderno e responsivo</li>

# <h3>🛠️ Tecnologias utilizadas<h3>

<li>HTML5 → Estrutura da aplicação</li>
<li>CSS3 → Estilização e layout</li>
<li>JavaScript (Vanilla JS) → Lógica, interatividade e manipulação do DOM</li>
<li>Local Storage → Persistência de dados no navegador</li>

# <h3>🎯 Objetivo do projeto<h3>

O principal objetivo foi praticar e consolidar conhecimentos em:

<li>Criação de interfaces interativas</li>
<li>Manipulação de eventos com JavaScript</li>
<li>Desenvolvimento de sistemas de autenticação no front-end</li>
<li>Organização de layout com sidebar</li>
<li>Armazenamento e tratamento de dados no navegador</li>

# <h3>🔐 Segurança</h3>

As informações de login e senha são armazenadas no Local Storage com criptografia básica.

⚠️ Observação: este método é utilizado apenas para fins educacionais e não substitui práticas seguras de autenticação em produção.
